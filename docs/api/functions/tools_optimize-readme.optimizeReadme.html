      Optimizes README content by restructuring, condensing, and extracting detailed sections.

      Performs intelligent README optimization including length reduction, structure improvement, content extraction to separate documentation, and TL;DR generation. Uses different strategies based on target audience (community, enterprise, developer, general) to maximize effectiveness.

      Parameters

      • input: Partial<OptimizeReadmeInput>

        The input parameters for README optimization

        • readme_path

          The file system path to the README file to optimize

        • strategy

          The optimization strategy to apply (default: "community_focused")

        • max_length

          Target maximum length in lines (default: 300)

        • include_tldr

          Whether to generate a TL;DR section (default: true)

        • preserve_existing

          Whether to preserve existing content structure (default: false)

        • output_path

          Optional output path for optimized README

        • create_docs_directory

          Whether to create docs/ directory for extracted content (default: true)

      Returns Promise<
          MCPToolResponse<
              { optimization: OptimizationResult; nextSteps: string[] },
          >,
      >

      Promise resolving to README optimization results

      Throws

      When README file is inaccessible or invalid

      Throws

      When optimization processing fails

      Throws

      When output directory cannot be created

      Example

      // Optimize README for community contributors
      const result = await optimizeReadme({
        readme_path: "./README.md",
        strategy: "community_focused",
        max_length: 300,
        include_tldr: true
      });

      console.log(`Reduced from ${result.data.optimization.originalLength} to ${result.data.optimization.optimizedLength} lines`);
      console.log(`Reduction: ${result.data.optimization.reductionPercentage}%`);

      // Optimize for enterprise with aggressive reduction
      const enterprise = await optimizeReadme({
        readme_path: "./README.md",
        strategy: "enterprise_focused",
        max_length: 200,
        preserve_existing: true
      });
